# Giuliano Cisneros (Haztuagencia) - Paquete Dream 100

Paquete de marketing completo construido para [Giuliano Cisneros](https://www.haztuagencia.com/) por [Zytrom](https://github.com/henryqueroa).

## Dos funnels en vivo

- **VSL Call Funnel** (`landing-page.html`): página de ventas larga con video de ventas (VSL) arriba y CTA a aplicar para una llamada. Voz match Haztuagencia.
- **Funnel de webinar** (`webinar.html`): página de registro para una clase en vivo gratis, con guion y secuencia de emails.

## Contenido

- **`index.html`** - Página hub con todos los entregables en accordion
- **`landing-page.html`** - VSL Call Funnel (sales page)
- **`webinar.html`** - Página de registro del webinar
- **`email-sequence.md`** - Secuencia de outreach de 10 emails (a Giuliano)
- **`webinar-script.md`** - Guion completo del webinar (tipo Perfect Webinar)
- **`webinar-emails.md`** - Secuencia de 5 emails del webinar (registro a cierre)
- **`loom-script.md`** - Script evergreen para Loom de 2 a 3 minutos
